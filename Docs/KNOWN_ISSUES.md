\# Known Issues



\- Visual Studio “Unable to build while Live Coding is active”

&nbsp; Решение: закрыть UE Editor или выключить Live Coding при сборке.



\- Иногда BP не показывает новые UFUNCTION после сборки

&nbsp; Решение: закрыть UE → Rebuild → открыть UE → Refresh All Nodes.



\- Warning про SkeletalMesh (deprecated API)

&nbsp; Не критично сейчас, но лучше позже обновить код на новый API UE.
