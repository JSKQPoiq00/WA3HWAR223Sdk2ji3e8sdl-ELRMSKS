\# RetroGame (UE 5.7)



Проект на Unreal Engine 5.7. Стиль: ретро/PSX. Основа: Third Person Story Adventure Template (TPSAT).

Цель: персонаж подбирает оружие, экипирует через инвентарь/быстрый доступ, стреляет/дерётся, сражается с зомби.



\## Требования

\- Unreal Engine 5.7

\- Visual Studio (у тебя стоит VS 2026 Insiders) с C++ workload

\- Git LFS (обязательно)



\## Как скачать проект

> ВАЖНО: нужен Git LFS



```bash

git clone <repo-url>

cd <repo-folder>

git lfs pull
